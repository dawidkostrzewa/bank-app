Bank Application

## Frontend

Technology stack:

1. **ReactJS** with **React-Router**
2. **Redux.js** 
3. **Material UI** framework

## Backend

Technology stack:

1. **NodeJS** with **Express.js** framework
2. **Sequelize** ORM
3. **MySQL** database
4. **RESTful API** with **JWT** authorization
